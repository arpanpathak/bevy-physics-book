## 📖 About

This book teaches **game physics mathematics** from the ground up using **Rust** and the **Bevy game engine**. Every concept is explained with:

- 🧮 Clear mathematical foundations
- 🦀 Idiomatic Rust code with generous comments
- 🎮 Bevy ECS architecture patterns
- 📊 ASCII diagrams for visual learners
- 🏗️ Progressive building — from vectors to a complete physics sandbox
